Traits (5.4)

A Trait is similar to a class, but only intended to group functionality in a fine-grained and consistent way

  • Build on one another
  • Inherited by child classes
  • Share common functionality
    • Varien_Object could be Varien_Trait

Traits (5.4)

                    <?php
                    trait Varien_Trait {
                        public function setData($key, $value = null) {
                            $this->data[$key] = $value;
                            return $this;
                        }
                        public function getData($key) {
                            return $this->data[$key];
                        }
                    }

Traits (5.4)

                    <?php
                    class SomeObject {
                        use Varien_Trait;
                        public function someFunction() {
                            $this->setData('Silly Example');
                        }
                    }

Generators (5.5)

  • Simple forward only iterators
  • Don't have to implement Iterator interface
  • A lot less memory is required!
    • Doesn't require the array to be built in memory
    • Simply yield the value

Generators (5.5)

                    class FizzBuzz {
                        public function getValues($num) {
                            for ($i = 1; $i <= $num; $i++) {
                                $value = ($i % 3) ? '' : 'Fizz';
                                $value .= ($i % 5) ? '' : 'Buzz';
                                yield $value ?: $i;
                            }
                        }
                    }

Generators (5.5)

                    <?php
                    $fb = new FizzBuzz();
                    foreach ($fb->getValues(1000000) as $val) {
                        echo $val. PHP_EOL;
                    }

PHP and Magento - Performance

Hip Hop

Hip Hop for PHP (HPHPc)

  • Released by Facebook in February 2010
  • Source to source compiles PHP to C++
  • Facebook reported up to 6x performance gain
  • Development process is very different
    • Compile each change
    • HPHPi - Custom interpreter for development
  • Discontinued in 2013 in favour of ...

Hip Hop Virtual Machine (HHVM)

  • JIT Virtual Machine for PHP and Hack languages
    • Hack is a dialect of PHP created by Facebook
    • Allows types to be specified for function arguments, return values.
  • 20 of the top PHP projects on GitHub pass
  • Simplifies development as HPHPi is no longer required
  • Used by Facebook, Wikipedia, Baidu, Etsy
